Python é uma linguagem puramente interpretada e não envolve nenhuma etapa de compilação.

Mentira!Embora seja conhecida como uma linguagem interpretada, o código-fonte Python (.py) é primeiro compilado para um formato intermediário chamado Bytecode (.pyc). É a Máquina Virtual do Python (PVM) que depois interpreta esse bytecode.
